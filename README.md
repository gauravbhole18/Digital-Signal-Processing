# 📡 Digital Signal Processing (DSP) Code Repository

This repository contains a collection of codes written as part of my **Digital Signal Processing (DSP)** coursework. The implementations cover fundamental DSP concepts and algorithms and can be used as reference material for learning or practice.

## Topics Covered

- Signal generation
- Discrete Fourier Transform (DFT)
- Fast Fourier Transform (FFT)
- Convolution
- Autocorrelation
- Basic signal analysis

## Purpose

The codes in this repository are meant to serve as **simple reference implementations** for students learning DSP concepts. They demonstrate how common DSP operations can be implemented programmatically.

## Usage

You can browse the scripts to understand how each algorithm works or run them to observe the results and outputs.
